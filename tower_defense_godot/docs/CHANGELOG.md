# 開發修改紀錄

本文記錄目前專案的重要修改，方便之後回頭查「改了什麼、為什麼改」。

## Beta 0.7.2 - 建造確認文字亂碼修正

- 版本號從 `Beta 0.7.1` 更新為 `Beta 0.7.2`。
- 重新產生 `fonts/NotoSansTC-Regular.ttf` 字型子集。
- 補入 `建造二次確認`、`正在建造中`、`請稍候`、`半透明塔預覽`、`進度條` 等新增 UI 與提示用字。
- 修正 Web 版右側 checkbox 與建造二次確認提示詞中文字亂碼。
- 字型仍維持子集化，大小約 `547KB`，避免回到完整中文字型造成載入變慢。

## Beta 0.7.1 - 建造確認範圍與塔種建造時間

- 版本號從 `Beta 0.7.0` 更新為 `Beta 0.7.1`。
- 修正二次確認時只能點塔左上角格的問題。
- 現在第二次點半透明 `2x2` 預覽範圍內任一格，都會確認建造該塔。
- 建造時間改為依塔種不同：箭塔 `0.2` 秒、砲塔 `0.3` 秒、冰凍塔 `0.5` 秒。
- 更新 `README.md`、`docs/GAME_LOGIC.md` 與 `docs/ROADMAP.md` 的版本與建造時間說明。

## Beta 0.7.0 - 半透明塔預覽與建造時間

- 版本號從 `Beta 0.6.9` 更新為 `Beta 0.7.0`。
- 右側操作區 `建造確認` 改名為 `建造二次確認 B`。
- 新增快捷鍵 `B`，可切換建造二次確認。
- 第一次點空地時，改為顯示目前選擇塔種的半透明塔預覽。
- 第二次點同一格後不再立刻出塔，改為進入 `0.5` 秒建造時間。
- 建造中會在建造區上方顯示進度條。
- 建造完成時會再次檢查金幣、佔地、敵人、路徑是否合法，合法才正式建塔。
- 切換塔種、開始下一波、拆塔、離開遊戲畫面時會取消目前預覽或建造中狀態。

## Beta 0.6.9 - 全域建造二次確認

- 版本號從 `Beta 0.6.8` 更新為 `Beta 0.6.9`。
- 建造改為預設全平台二次確認，不再依賴 Web 或 Godot 是否能正確分辨手機設備。
- 第一次點空地只顯示半透明 `2x2` 預覽，第二次點同一格且可建造時才真正建造塔。
- 點到已有塔時仍直接選取塔，保留快速操作。
- 觸控後仍會短暫忽略模擬滑鼠點擊，避免同一次觸控被重複處理。
- 右側操作區新增 `建造確認` checkbox，取消勾選可改回點一下直接建造。
- 不可建造位置會顯示紅色預覽並播放錯誤提示音。
- 新增 `BuildManager.validate_build()`，集中建造合法性檢查，讓滑鼠建造、觸控預覽與真正建造共用同一套規則。
- 更新 `README.md`、`docs/GAME_LOGIC.md` 與 `docs/ROADMAP.md` 的版本與建造確認說明。

## Beta 0.6.1 - Netlify 載入速度與快取優化

- 版本號從 `Beta 0.6.0` 更新為 `Beta 0.6.1`。
- Netlify build 匯出 Web 版後，會將核心資源改成版本化檔名：
  - `td-<hash>.js`
  - `td-<hash>.wasm`
  - `td-<hash>.pck`
  - `td-<hash>.audio.worklet.js`
  - `td-<hash>.audio.position.worklet.js`
- 自動改寫 `index.html`，讓 Godot Web 入口載入版本化資源。
- build 後為核心資源產生 `.gz` 壓縮檔；若環境有 `brotli`，也會產生 `.br`。
- 新增 Netlify headers：
  - `index.html` 不長快取，確保玩家每次進站會拿到最新入口。
  - `td-*`、`.wasm`、`.pck` 等版本化資源設定一年長快取與 `immutable`。
  - `.wasm` 與 `.pck` 補上對應 Content-Type。
  - 保留 WebAssembly / Godot Web 需要的跨源隔離 headers。

  - 路徑搜尋、防堵路、斜線防穿角。
- `scripts/game_renderer.gd`
  - 繪圖集中管理。
  - 格線、路徑、塔、敵人、子彈、背景、Game Over。
- `scripts/game_ui.gd`
  - UI 建立、按鈕 signal、版面配置、HUD 更新。
- `scripts/game_data.gd`
  - 塔與敵人資料表讀取。
- `scripts/wave_manager.gd`
  - 難度、起始資源、敵人倍率、出怪規則、波次流程。
- `scripts/save_manager.gd`
  - 存檔資料組裝、JSON 讀寫、meta 讀寫。
