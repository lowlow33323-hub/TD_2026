# 開發修改紀錄

本文記錄目前專案的重要修改，方便之後回頭查「改了什麼、為什麼改」。

## Beta 0.5.3 - 鍵盤快捷鍵擴充

- 版本號從 `Beta 0.5.2` 更新為 `Beta 0.5.3`。
- 新增 `I`：將目前選取塔升級到最高等，並逐級檢查金幣是否足夠。
- 新增 `D`：加快速度，依序切換 `1x -> 2x -> 4x -> 16x`。
- 新增 `S`：降低速度，依序切回 `16x -> 4x -> 2x -> 1x`。
- 新增 `R`：切換是否顯示敵人路徑。
- 新增 `A`：切換自動開始。

## Beta 0.5.2 - 升級星星、受擊弧與塔地板

- 版本號從 `Beta 0.5.1` 更新為 `Beta 0.5.2`。
- 5 級塔標記改為純色填滿橘色五角星。
- 子彈命中衝擊波改為 120 度弧形。
- 衝擊波會顯示在受擊點的子彈反方向，並沿用不同塔的攻擊顏色。
- 塔佔用區塊改為更深色地板。
- 塔佔用區塊不再顯示格線。

## Beta 0.5.1 - 路徑重繪、受擊衝擊波與音量控制

- 版本號從 `Beta 0.5.0` 更新為 `Beta 0.5.1`。
- 敵人路徑顯示改為一次繪製完整連續折線，避免被切成虛線。
- 敵人受到子彈攻擊時會顯示擴散衝擊波。
- 背景音樂改為更有戰鬥感的節奏循環。
- 背景音樂預設音量提高約 50%。
- 右側資訊欄新增 `音樂` 與 `音效` checkbox。
- `音樂` 與 `音效` 後方新增音量拖拉控制。

## Beta 0.5.0 - 路徑轉角疊色修正

- 版本號從 `Beta 0.4.9` 更新為 `Beta 0.5.0`。
- 修正敵人路徑顯示在轉彎處半透明圖形互相重疊、顏色變深的問題。
- 路徑線段會在轉角前後縮短，轉角只保留單次同色圓滑銜接。

## Beta 0.4.9 - 子彈簡化、生命提示與塔瞄準

- 箭塔發射臂會追蹤目前目標旋轉。

## Beta 0.4.8 - 40 波後敵群、路徑透明與箭矢方向

## Beta 0.3.1 - 架構整理與平衡調整

### 專案搬移與基礎確認

- 確認 Godot 專案可在目前電腦開啟與執行。
- 確認目前主要開發專案為：
  - `tower_defense_godot`
- 後續不再更新舊的 `path-bender-tower-defense-(4.4)` 副本。

### 畫面與 UI 調整

- 遊戲畫面改為依視窗大小自適應。
- 增加主選單頁面。
- 主選單包含：
  - 操作規則
  - 排行榜
  - 開始遊戲
  - 難度調整
  - 繼續遊戲
- 遊戲中新增 UI 操作：
  - 存檔
  - 回主選單
  - 退出
  - 速度切換
- `存檔` 按鈕改成只有波與波之間可用。
- `回主選單`、`退出` 增加二次確認視窗。
- 修正 `game_panel` 因 label autowrap 導致資訊框異常撐高、遮住建造區的問題。

### 地圖與建造規則

- 建造區改為 `30x20`。
- 塔佔地改為 `2x2`。
- 建塔前會檢查是否會完全堵死入口到出口。
- 敵人進行中不能建造新塔。
- 滑鼠 hover 到建造區時，顯示半透明 2x2 建造預覽。
- 可建造顯示藍色，不可建造顯示紅色。
- 嘗試蓋在不可建造區塊時，播放錯誤提示音。

### 波次與玩法

- 敵人移動改為可走斜線。
- 斜線移動加入防穿角限制，避免敵人穿過被塔堵住的角落。
- 第 50 波設定為最後一關。
- 按下 `下一波` 時增加提示音。
- 出生點增加發光特效。
- 增加 Boss 波與 Boss 擊殺排行榜。

### 架構拆分

從原本較大的 `main.gd` 拆出多個職責檔案：

- `scripts/game_defs.gd`
  - 共用常數。
  - 地圖大小、塔類型、敵人類型、難度、畫面狀態、版本號。
- `scripts/tower.gd`
  - 塔資料、升級、數值套用。
- `scripts/enemy.gd`
  - 敵人資料、血量、速度、狀態效果。
- `scripts/projectile.gd`
  - 子彈資料、傷害、緩速、濺射參數。
- `scripts/pathfinder.gd`
  - 路徑搜尋、防堵路、斜線防穿角。
- `scripts/game_renderer.gd`
  - 繪圖集中管理。
  - 格線、路徑、塔、敵人、子彈、背景、Game Over。
- `scripts/game_ui.gd`
  - UI 建立、按鈕 signal、版面配置、HUD 更新。
- `scripts/game_data.gd`
  - 塔與敵人資料表讀取。
- `scripts/wave_manager.gd`
  - 難度、起始資源、敵人倍率、出怪規則、波次流程。
- `scripts/save_manager.gd`
  - 存檔資料組裝、JSON 讀寫、meta 讀寫。

### 資料化

- 新增 `data/towers.json`
  - 塔名稱、價格、射程、攻速、傷害、濺射、緩速、顏色。
- 新增 `data/enemies.json`
  - 敵人名稱、血量倍率、速度倍率、擊殺金幣、顏色。
- `tower.gd` 改為從 `GameData` 讀取 `towers.json`。
- `enemy.gd` 改為從 `GameData` 讀取 `enemies.json`。
- `main.gd` 與 `game_ui.gd` 的塔名稱/價格改為查資料表，避免顯示與實際數值不同。

### 常數整理

- `main.gd` 移除與 `game_defs.gd` 重複的常數。
- `main.gd` 改為使用：

```gdscript
const Defs = preload("res://scripts/game_defs.gd")
```

- 例如：
  - `TYPE_CANNON` -> `Defs.TYPE_CANNON`
  - `SCREEN_MENU` -> `Defs.SCREEN_MENU`
